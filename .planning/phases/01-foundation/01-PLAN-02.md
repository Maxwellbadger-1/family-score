<objective>
Erstellt das lokale Swift Package FamilyScoreKit mit shared Codable+Sendable Datenmodellen und dem App Group Identifier als single source of truth.

Purpose: FamilyScoreKit ist die Schnittstelle zwischen App Target und Widget Extension. Ohne dieses Package müssten Datenmodelle in beide Targets kopiert werden (Target-Membership-Dopplung = Pitfall aus PITFALLS.md). Das Package enthält bewusst kein Supabase SDK — nur Plain Swift + Foundation — damit das Widget Extension Target keine Supabase-Abhängigkeit zieht.

Output:
- FamilyScoreKit/Package.swift (SPM-Manifest, keine externen Deps)
- FamilyScoreKit/Sources/FamilyScoreKit/WidgetData.swift (WidgetData struct, appGroupIdentifier, placeholder)
</objective>

```swift
// Sources/FamilyScoreKit/WidgetData.swift
// Shared between App Target and Widget Extension
// KEIN Supabase SDK import — nur Foundation
import Foundation

public struct WidgetData: Codable, Sendable {
    public struct MemberScore: Codable, Sendable {
        public let displayName: String
        public let avatarInitial: String
        public let weeklyPoints: Double
        public let weeklyMinutes: Int

        public init(displayName: String, avatarInitial: String,
                    weeklyPoints: Double, weeklyMinutes: Int) {
            self.displayName = displayName
            self.avatarInitial = avatarInitial
            self.weeklyPoints = weeklyPoints
            self.weeklyMinutes = weeklyMinutes
        }
    }

    public let familyName: String
    public let members: [MemberScore]
    public let lastUpdated: Date

    public init(familyName: String, members: [MemberScore], lastUpdated: Date) {
        self.familyName = familyName
        self.members = members
        self.lastUpdated = lastUpdated
    }
}

// App Group Identifier — single source of truth für App und Widget
public let appGroupIdentifier = "group.com.familyscore"

// Placeholder für Widget-Previews und Snapshots
extension WidgetData {
    public static let placeholder = WidgetData(
        familyName: "Familie Muster",
        members: [
            MemberScore(displayName: "Max", avatarInitial: "M",
                        weeklyPoints: 120, weeklyMinutes: 90),
            MemberScore(displayName: "Anna", avatarInitial: "A",
                        weeklyPoints: 95, weeklyMinutes: 75)
        ],
        lastUpdated: Date()
    )
}
```

<threat_model>
## Trust Boundaries

| Boundary | Beschreibung |
|----------|-------------|
| FamilyScoreKit Package → Widget Extension | Package darf keine Netzwerk-Capabilities oder Supabase SDK enthalten — Widget Extension hat 30MB Limit |
| App Group UserDefaults | Einziger erlaubter Kanal für Daten zwischen App und Widget; FamilyScoreKit.appGroupIdentifier ist single source of truth |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-1-03 | Denial of Service | FamilyScoreKit/Package.swift | mitigate | Package.swift hat keine externen Dependencies; `grep supabase Package.swift = 0` als Acceptance Criterion verhindert versehentliche SDK-Einbindung |
| T-1-06 | Tampering | appGroupIdentifier Konstante | mitigate | Single source of truth in FamilyScoreKit — beide Targets importieren dieselbe Konstante; Typo-Fehler würden Widget-Lesen von falscher Gruppe verursachen; Acceptance Criterion prüft exakten String "group.com.familyscore" |
| T-1-07 | Information Disclosure | WidgetData struct | accept | WidgetData enthält nur Display-Daten (Name, Punkte, Minuten) — keine PII, keine Auth-Tokens; Serialisierung via JSONEncoder in App Group (OS-Sandbox geschützt) |
</threat_model>
